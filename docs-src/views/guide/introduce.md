# 介绍

### Cloud-UI 是什么？

Cloud-UI 是网易云计算前端团队推出的一款基于 Vue.js 2.0 的桌面端 UI 组件库，主要服务于网易云官网及其相关的后台产品。

在做某个项目的时候，我们通常会把项目中的通用组件放在一个地方进行统一管理，这便是组件库的由来。Cloud-UI 就是我们团队（网易云计算技术部）经过大量的项目实践和总结，逐步形成的一套通用组件库。通过模块化的解决方案，可以降低开发成本，为用户提供更好的用户体验。

### 特性

- 高质量、功能丰富
- 开箱即用、扩展灵活
- 持续维护更新

### 支持环境

- 支持 Vue.js 2.x
- 支持现代浏览器和 IE9 及以上
