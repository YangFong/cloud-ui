# 线型进度条 LinearProgress

## 示例
### 基本形式

``` html
<u-linear-progress :percent="36"></u-linear-progress>
```

## API
### Props/Attrs

| Prop/Attr | Type | Default | Description |
| --------- | ---- | ------- | ----------- |
| percent | Number | `0` | 百分比 |
| size | String | `'normal'` | 高度扩展，值有`'small'`, `'normal'`, `'large'`, `'huge'`四个值 |

### Slots

#### (default)

插入文本或HTML。
