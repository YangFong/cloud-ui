### 基本用法

``` html
<u-tabs value="B">
    <u-tab title="选项卡A" value="A">选项卡 A 的内容</u-tab>
    <u-tab title="选项卡B" value="B">选项卡 B 的内容</u-tab>
    <u-tab title="选项卡C" value="C">选项卡 C 的内容</u-tab>
</u-tabs>
```
