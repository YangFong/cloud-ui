# Logo

## 示例
### 基本形式

``` html
<div style="background: #4289db; padding: 0 10px;">
    <u-logo></u-logo>
</div>
```

### 子标题

``` html
<div style="background: #4289db; padding: 0 10px;">
    <u-logo><a href="/" target="_blank">ICP 备案系统</a></u-logo>
</div>
```

### 颜色扩展

``` html
<div style="background: #343f4f; padding: 0 10px;">
    <u-logo color="colorful"></u-logo>
</div>
```
